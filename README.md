## 项目简介

InteractiveChinaMap 是一个交互式中国地图学习平台，旨在通过游戏化的方式帮助用户掌握中国地理知识，包括省份名称、地理位置、省份轮廓、城市位置以及边界等。平台提供了多种学习模式，适合不同水平的学习者。

## 功能特性

- **首页入口**：支持多语言切换、用户登录注册、管理员入口。
- **模式选择**：提供拼图模式、挑战模式、学习模式等多种选择。
- **省份拼图**：根据提示在地图上点击对应的省份。
- **城市拼图**：根据提示在地图上点击对应的城市。
- **挑战模式**：计时答题，测试地理知识掌握程度。
- **每日挑战**：每日更新的答题挑战，记录历史成绩。
- **学习模式**：交互式地图，鼠标悬停查看省份/城市信息。
- **标准地图**：全面的中国地理信息参考，包含省份边界、人口、GDP 等简介。
- **成就系统**：完成特定任务解锁成就，展示学习进度。
- **统计分析**：查看个人答题正确率、连续打卡天数等统计数据。
- **反馈系统**：提交使用反馈和改进建议。

## 技术栈

- **后端**：Node.js (原生 http 模块)
- **前端**：HTML5, CSS3, JavaScript (原生)
- **数据存储**：JSON 文件
- **地图资源**：SVG 矢量图

## 快速开始

### 前提条件

- Node.js (v12.0.0 或更高版本)

### 安装

1. 进入后端目录：
   ```bash
   cd backend
   ```

2. 安装依赖（如果 package.json 中有依赖）：
   ```bash
   npm install
   ```

3. 启动服务：
   ```bash
   node server.js
   ```

4. 打开浏览器访问：`http://localhost:3000` (默认端口)

## 如何使用

1. **注册/登录**：在首页点击登录/注册按钮创建账号。
2. **选择模式**：进入模式选择页面，选择你想要练习的类型。
3. **开始学习/挑战**：按照页面提示进行操作。
4. **查看统计**：在个人中心查看学习进度和成就。
